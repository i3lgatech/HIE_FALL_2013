README
======

.NET Restful Web App project
============================

1. Download the folder HIE_group_3\WebApp_.NET and use Visual Studio 2012 to open the project
2. Publishes four RESTful APIs to connect with the Greenway API Server and retrieve patient files

APITests_Java
=============

1. Java client to test the Restful APIs published
2. Download the folder HIE_group_3\APITests_Java and use Eclipse 4.x to open the project

Policy_Snapsflow
================

1. Snapsflow policy to retrieve patient CCD and upload into the MS HealthVault
2. Import the HIE_group_3\Policy_Snapsflow\PolicyExport.xml in Snapsflow
